# Web 开发者转 iOS 开发教程 - UIKitDemoAPP 

#### 介绍
UIKitDemoApp for learning UIKit


#### 软件架构
软件架构说明

#### 前提要求
1. 安装 CocoaPods

#### 安装教
1.  pod install
2.  打开 UIKitDemoApp.xcworkspace 项目
3.  运行构建，模拟器选择 iPhone-13 以上
